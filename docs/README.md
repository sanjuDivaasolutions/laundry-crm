## Quick Reference

### Technology Stack
- **Backend**: Laravel 9 (PHP 8.1+)
- **Frontend**: Vue 3 with TypeScript
- **UI Framework**: Metronic 8 (Bootstrap 5.2.2)
- **State Management**: Pinia
- **Build Tool**: Vite
- **Authentication**: JWT (php-open-source-saver/jwt-auth)
- **Database**: MySQL with Model Caching
- **File Storage**: Spatie Media Library

### Key Features
- Multi-tenant ERP system for sign rental business
- Comprehensive inventory management with warehouse & shelf tracking
- Sales & purchase order processing
- Contract management with Stripe integration
- Advanced reporting system
- Multi-language support
- Role-based permissions
- PDF generation and Excel import/export

### Development Commands
```bash
# Backend
php artisan serve
php artisan test
composer install

# Frontend
npm run dev
npm run build
npm install

# Database
php artisan migrate
php artisan db:seed
```
